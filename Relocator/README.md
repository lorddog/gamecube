# Gamecube - Relocator 2.04
```
Reloc 2.04 - The Better Way - StarCube
======================================
Usage:  Reloc <Ripped Image> [Image name or Extract Options] -Options

Options are:
============
 -r     Relocate Image - Files will get new Position, FST will be touched
 -f     Extract Original FST also
 -d     Disable Debug/Progress Diplay Output
 -n     Disable Image-Size-Fix (Image will be keep only needed size only)
 -nofix Disable AutoFix Game Image (If Needed)

Extract Options are:
====================
 -bi2   Extract BI2.bin
 -md    Extract Main.Dol
 -ap    Extract Apploader
 -fst   Extract Original FST


 Screen-Chars: X = New File begin's
               . = Data of File (X)
               O = One File Relocated (-r Only!)


```
